## 🚀 Déploiement

### Étapes Suivantes

1. **Build Next.js**
   ```bash
   npm run build
   ```

2. **Test Local**
   ```bash
   npm run dev
   ```
   Vérifier quelques pages :
   - http://localhost:3000/epaviste/lyon/
   - http://localhost:3000/epaviste/versailles/

   ```bash
   npm run deploy

   vercel deploy --prod
   ```

4. **Vérification SEO**
   - Google Search Console : Soumettre sitemap
