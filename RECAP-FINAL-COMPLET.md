## 🚀 Prochaines étapes

### Immédiat
1. **Installer Node 20+** (requis pour Next.js)
   ```bash
   brew install node@20
   # ou
   nvm install 20 && nvm use 20
   ```

2. **Lancer en local**
   ```bash
   cd reseau-epaviste
   npm install
   npm run dev
   ```
   Ouvrir : http://localhost:3000

3. **Remplacer numéro factice**
   ```bash
   find src -type f -name "*.tsx" -exec sed -i '' 's/01 84 17 32 18/VOTRE_NUMERO/g' {} +
   ```

4. **Build production**
   ```bash
   npm run build
   ```

5. **Déployer**
   ```bash
   vercel --prod
   # ou autre plateforme
   ```

6. **Google Search Console**
   - Soumettre sitemap
   - Demander indexation 10-15 pages prioritaires
